Here's how we do links at the foundation.

Google Parameters

We use the following five parameters for url tracking (google analytics). If the url has a question mark anywhere in it you want to add &utm_source=something&utm_medium=something&utm_campaign&utm_content=something&utm_term=something. If it doesn't you want to add
?utm_source=something&utm_medium=something&utm_campaign&utm_content=something&utm_term=something
to the end.

  • source: platform that the email is coming from. (Use eblast for emails)
  • medium: type of content. (use email for emails that come out of Luminate)
  • content: Descriptor for the specific email. (Ex. Scotiabank)
  • campaign: Internal Campaign for how this/these emails are being grouped. (Ex. Angel)
  • term: Description of what is being used as the link. We do this so we're able to distinguish different links to the same website in an email. (ex. Header_Button)
